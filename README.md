A high-performance, modern landing page for the **Konya Technical University Artificial Intelligence and Data Science Laboratory (KTUN AI Lab)**. This project serves as the digital headquarters for the laboratory's "Innovation 2.0" transition, showcasing advanced research, achievement tracking, and a multidisciplinary ecosystem.

## 🚀 Technical Excellence

This project is built using the latest industry standards for web development, focusing on performance, scalability, and high-end aesthetics.

### Tech Stack
- **Framework:** [Next.js 14](https://nextjs.org/) (App Router Architecture)
- **Language:** [TypeScript](https://www.typescriptlang.org/) for robust type safety and developer productivity.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with a custom **Glassmorphism Design System**.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for complex, high-performance GPU-accelerated UI interactions.
- **Icons:** [Lucide React](https://lucide.dev/) for consistent, lightweight vector iconography.
- **Theming:** `next-themes` for seamless dark/light mode transitions.

## 🏗️ Project Structure

The codebase follows a modular, scalable architecture using the `src` directory pattern:

```text
src/
├── app/               # Next.js App Router (Layouts, Pages, Global Styles)
├── components/        # Highly reusable, atomic UI components
│   ├── hero.tsx       # Dynamic landing section with Framer Motion
│   ├── team.tsx       # Management & Academic advisor modules
│   ├── achievements.tsx# Project & Award display logic
│   └── ...            # Specialized sections (Vision, Cooperation, etc.)
└── ...                # Configuration and asset management
```

## ✨ Key Features

- **Innovation 2.0 Branding:** A premium visual language emphasizing AI, data science, and futuristic research.
- **Responsive Architecture:** Fully fluid design optimized for everything from ultra-wide monitors to mobile devices.
- **Interactive UI:** Advanced entrance animations, hover states, and organic scrolling patterns powered by Framer Motion.
- **Performance Optimized:** Utilizing Next.js Server Components (RSC) where possible and `use client` selectively for interactivity.
- **Theme Support:** Intelligent theme detection and persistence.
